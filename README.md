# Lógica de Programación

Este repositorio contiene ejercicios y material de apoyo para practicar los fundamentos de la lógica de programación.

## Ejercicios
- Desarrolla un programa donde insertes 2 numeros y verifiques cual es el mayor, muestra por consola el resultado
- Simula un inicio de sesion donde pidas usuario y contrasena, si el usuario es incorrecto muestra un mensaje que diga usuario incorrecto y si es correcto muestra un mensaje que diga usuario correcto
- Pide por consola un un numero y mediande un ciclo for imprimero continuamente
- Imprime en pantalla el resultado de una funcion que pida cualquier tipo de texto e imprimerlo
- Desarrolla con switch/case donde cada opcion te muestre un proceso distinto, como validar un numero negativo, mostrar una lista de datos o mostrar en pantalla un mensaje en consola.
